Glosario

Frontend
Parte visual del sistema que interactúa con el usuario (por ejemplo, una página web con horarios).

Backend
Lógica interna del sistema que procesa la información y se comunica con la base de datos.

Base de datos
Sistema donde se almacena y organiza la información de profesores, horarios, materias y aulas.

Wordpress
Sistema de gestión de contenidos (CMS) que permite crear y administrar sitios web de forma sencilla, sin necesidad de conocimientos avanzados de programación.

Plugins
Extensión o complemento que se instala en WordPress para añadir nuevas funcionalidades al sitio, como formularios, carritos de compra, optimización SEO, entre otros.
